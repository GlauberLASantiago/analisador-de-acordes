# Analisador de Acordes Tonais

Aplicativo em HTML único gerado a partir das planilhas fornecidas. Basta abrir `analisador_de_acordes.html` no navegador ou publicar no GitHub Pages.

Arquivos:
- `analisador_de_acordes.html`: aplicativo completo, sem dependências externas.
- `base_acordes.csv`: exportação da base de análise em formato plano, para auditoria/edição.

Observação: a planilha-base contém análise completa para tétrades maiores. Para tríades e 7sus4, o app usa uma adaptação conservadora a partir das famílias correspondentes de tétrades e exibe aviso no resultado.
